# ⚽ FC San Bernardo - Colección Oficial

¡Bienvenido al álbum virtual del **FC San Bernardo**! Esta es una aplicación web interactiva donde los jugadores y fans pueden coleccionar cromos exclusivos utilizando códigos secretos.

## 🚀 Características
- **Revelación de cromos:** Ingresa un código (como `SB4`) para ver a tu jugador favorito.
- **Efectos especiales:** ¡Prueba el código `D1` para una sorpresa dorada! 🎊
- **Base de datos dinámica:** Los jugadores se gestionan mediante un archivo JSON.

## 🛠️ Tecnologías utilizadas
- **Backend:** Node.js (Servidor HTTP nativo)
- **Frontend:** HTML5, CSS3 y JavaScript vanilla
- **Efectos:** Canvas Confetti API
- **Hosting:** Render.com

## 📖 Cómo usar
1. Entra al link de la aplicación (proporcionado por el administrador).
2. En el campo de texto, escribe el código de cromo que hayas obtenido.
3. Presiona el botón **REVELAR JUGADOR**.
4. ¡Si el código es correcto, aparecerá el cromo del jugador!
